# マルチサイト設定の確認方法

## 現在の設定状態

- `WP_ALLOW_MULTISITE`: `true` ✓
- `MULTISITE`: 定義されていない ✓（正しい状態）

## 確認手順

### 1. 管理画面にログイン

`http://localhost:8000/wp-admin` にアクセスしてログイン

### 2. 「ツール」メニューを確認

左メニューに「ツール」が表示されているか確認してください。

### 3. 「ネットワークの設置」を確認

「ツール」をクリックして、「ネットワークの設置」が表示されるか確認してください。

## 表示されない場合の確認事項

### プラグインを無効化

すべてのプラグインを無効化してください：

1. **プラグイン** → **インストール済みプラグイン**
2. すべてのプラグインを無効化

### ブラウザのキャッシュをクリア

1. ブラウザのキャッシュをクリア
2. または、シークレットモード/プライベートモードでアクセス

### ログアウトして再ログイン

1. 管理画面からログアウト
2. 再度ログイン

### wp-config.phpの確認

以下のコマンドで確認：

```bash
php -r "require 'wp-config.php'; echo defined('WP_ALLOW_MULTISITE') && WP_ALLOW_MULTISITE ? 'OK' : 'NG';"
```

`OK`と表示されれば設定は正しいです。

## データベースの状態

データベースには通常のWordPressテーブルのみが存在し、マルチサイト用のテーブル（`wp_blogs`, `wp_site`など）は存在しません。これは正常な状態です。

マルチサイトを有効化すると、WordPressが自動的にこれらのテーブルを作成します。
